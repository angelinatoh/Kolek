# Kolek

Kolek is a **Final Year Project** developed in collaboration with **PETRONAS Dagangan Berhad (PDB)**.
It is a **Smart Used Cooking Oil (UCO) Collection System** designed to improve the efficiency, transparency, and sustainability of UCO collection operations.

The system integrates mobile applications and a web dashboard to streamline communication between collectors, households, and administrators, while promoting environmentally responsible practices.

---

## 📱 System Modules

* **Collector Mobile App**
  Built using Flutter to manage assigned collection tasks, update collection status, and record UCO weights.

* **Household Mobile App**
  Allows users to request UCO collection, track requests, and engage with sustainability initiatives.

* **Admin Dashboard (Web)**
  A web-based platform for monitoring operations, managing users, and analyzing collection data.

---

## 🛠 Technologies Used

**Mobile Application:**

* Flutter

## 🌱 Key Features

* Real-time UCO collection tracking
* Task assignment and status management
* Weight calculation and reward system
* QR / code-based verification for collection completion
* Awareness and sustainability campaign integration

## 🎯 Project Objective

The goal of Kolek is to:

* Promote proper disposal of used cooking oil
* Reduce environmental pollution
* Provide a structured and digitalized UCO collection system
* Support sustainability initiatives aligned with industry practices
